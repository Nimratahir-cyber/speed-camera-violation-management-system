# Speed Camera Violation Management System
This is a C++ console-based application that simulates a speed camera system. It records vehicle speed violations, calculates fines based on speed limits, and manages payment status. The system also stores data persistently using file handling.

# Features
- Add vehicle violation records

- Automatically calculate fine based on speed limit

- View all violation records

- Search specific vehicle records

- Pay unpaid fines

- Calculate total unpaid fines

- Save and load records using file handling

# Speed Limits
- City Road:   60 km/h
  
- Highway:     100 km/h

- Motorway:    180 km/h

# How to run
1. Compile the program using a C++ compiler (g++ recommended)

2. Run the executable file

3. Follow the on-screen menu options
